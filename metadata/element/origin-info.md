{: .d-block .mt-1 }

#### Guidelines
{: .no_toc }
- Select the appropriate date **Type**
  - Use **Date created** for most unpublished materials
  - Use **Date issued** for most published materials
  - Do not use **Date captured**, **Copyright date**, or **Other date**
  - Dates related to ownership, copyright, gift, etc. can be expressed as free text in a **Note** element with the **Type** ‚Äúdate‚Äù with a prefix for context, i.e. ‚ÄúGift: 1937‚Äù
